Agentic AI Book Buddy

An AI-powered multi-agent novel recommendation system built using Google Agent Development Kit and Gemini.

View Kaggle Notebook View Code

The Problem

Many existing book recommendation systems rely on popularity or surface-level metadata. Book Buddy focuses on understanding why a reader loved their favourite books and recommends novels that match their themes, tone, and writing style.

How It Works

  • User provides three favourite novels
  • Preference extractor agent infers reading taste
  • Book search agent finds matching novels using Google Search
  • Orchestrator agent selects the best recommendation
  • Reading preferences are stored for future sessions

Architecture

  • preference_extractor_agent: Extracts genres, themes, tone, and pacing
  • book_search_agent: Searches the web for suitable novels
  • novel_recommender_orchestrator: Coordinates agents and generates final output

Tech Stack

  • Google Agent Development Kit (ADK)
  • Gemini 2.5 Flash Lite
  • Multi-agent orchestration
  • Google Search tool
  • Session state and long-term memory
  • Kaggle notebooks
